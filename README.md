# riba2534 ACM 模板

一份持续打磨的 ACM/ICPC 算法竞赛模板，收录数论、图论、数据结构、动态规划、字符串、计算几何等常用算法的实现代码与讲解。

## 📖 在线阅读

文档站：**[acm.riba2534.cn](https://acm.riba2534.cn)**

支持全文搜索、数学公式渲染与深浅色主题，使用 [VitePress](https://vitepress.dev) 构建。

## 📚 内容分类

- **常用技巧** —— 输入输出加速、STL、pb_ds、位运算、小技巧
- **数论** —— 欧拉函数、快速幂、逆元、素数筛、康托展开、博弈论
- **组合数学** —— 母函数、BM 线性递推、容斥原理、组合数、卢卡斯定理
- **图论** —— 最短路、最小生成树、网络流、二分图匹配、LCA、点分治、Tarjan
- **字符串** —— KMP、扩展 KMP、AC 自动机、Manacher、后缀数组、后缀自动机
- **数据结构** —— 线段树、树状数组、主席树、树链剖分、莫队、Trie、线性基
- **动态规划** —— 背包、LIS、LCS、编辑距离、区间/树形/数位/状压 DP
- **高精度运算** —— 大数的加减乘除
- **其他** —— 计算几何、表达式求值、归并排序、模拟退火、二分查找、各种公式

例题代码大多来源于作者的 [CSDN 博客](https://blog.csdn.net/riba2534)。也可在 [Releases](https://github.com/riba2534/my_acm_template/releases) 下载 PDF 版本。

## 🛠️ 本地开发

```bash
npm install      # 安装依赖
npm run dev      # 本地预览（http://localhost:5173）
npm run build    # 构建静态站点到 docs/.vitepress/dist
```
