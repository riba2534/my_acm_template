# 小技巧

### string构造函数

> string(原串，开始位置，开始位置后的长度)

例子:

```cpp
string s1="123456789";
string s2(s1,3,5);//代表:45678
```

## 字符串转int

```cpp
atoi();//普通char*型
atoi(str.c_str());//string类型
```

## int转string

```cpp
string get_s(int n)
{
    stringstream ss;
    string str;
    ss<<n;
    ss>>str;
    return str;
}
```

## define中`##`的用法

是直接连接的意思，有时候会起到意想不到的作用

```
#include <stdio.h>

#define WTF(x) 1##x

int main()
{
    printf("%d\n",WTF(3));//输出13,直接连接的意思
    return 0;
}

```

## 快排模板

```cpp
void qsort(int A[], int l, int r)
{
    if(l >= r) return ;
    int m = l-1;
    for(int i=l; i<=r; i++)
        if(A[i] <= A[r])
            swap(A[++m], A[i]);
    qsort(A, l, m-1);
    qsort(A, m+1, r);
}
```
