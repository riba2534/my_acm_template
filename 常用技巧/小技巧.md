# 小技巧

### string构造函数

> string(原串，开始位置，开始位置后的长度)

例子:

```cpp
string s1="123456789";
string s2(s1,3,5);//代表:45678
```

## 字符串转int

```cpp
atoi();//普通char*型
atoi(str.c_str());//string类型
```

## int转string

```cpp
string get_s(int n)
{
    stringstream ss;
    string str;
    ss<<n;
    ss>>str;
    return str;
}
```
