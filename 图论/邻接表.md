# 邻接表

邻接表一般可以用`vector`来实现，或者手写邻接表

通俗的写法是这样(链式前向星)

```cpp
const int N=10000+50;
int first[N],tot;
struct edge
{
    int u,v,w;
    int next;
} e[N*2];
void add_edge(int u,int v,int w)//加边
{
    e[tot].u=u;
    e[tot].v=v;
    e[tot].w=w;
    e[tot].next=first[u];
    first[u]=tot++;
}

{

    tot=0;
}
```
