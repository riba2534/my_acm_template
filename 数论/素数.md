# 素数相关/线性筛法

判断素数:

```cpp
int sushu(int x)
{
    if(x<=1)return 0;
    for(int i=2; i<=sqrt(x); i++)
        if(x%i==0)
            return 0;
    return 1;
}
```

线性筛法:

```cpp
const int N=1000+7;
ll  prime[N]= {1};
int pcnt=0;
int factor[N]= {1,1};
void Init_Prime()
{
    pcnt=1;
    for(ll i=2; i<N; i++)
    {
        if(!factor[i])
        {
            prime[pcnt++]=i;
            factor[i]=i;
        }
        for(ll j=1; j<pcnt&&i*prime[j]<N; j++)
        {
            factor[i*prime[j]]=prime[j];
            if(!(i%prime[j]))
                break;
        }
    }
    return;
}
 
```

普通筛法:

```cpp
const int N=2000100;
int isprime[N];
void init()
{
    int i,j;
    isprime[1]=1;
    for(i=2; i<=sqrt(N); i++)
    {
        if(!isprime[i])
        {
            for(j=i+i; j<=N; j+=i)
                isprime[j]=1;
        }
    }
}
```
